Snippetbox

Full-Stack Go Application

A full-stack Go web application called "Snippetbox" that lets users CRUD text snippets (similar to GitHub gists). Features include:

- Authentication. Users can register and sign in.

- Protected endpoints. Only signed-in users can create snippets.

- RESTful routing.

- Middleware.

- MySQL database.

- SSL/TLS web server using HTTP 2.0.

- Generated HTML via Golang templates.

- CRSF protection.
